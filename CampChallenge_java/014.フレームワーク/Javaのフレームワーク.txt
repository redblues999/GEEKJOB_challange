フレームワーク…プログラミングの雛形。アプリの開発を効率よく行うことができる。


JSF(Java Server Faces)
・Javaの標準仕様のフレームワーク
・ユーザーインターフェースを簡単に作ることができる
・コンポーネントベースのアプローチをとっている

コンポーネントベースの手順
表示する画面が決まる
↓
それに対応する画像処理クラス(バッキングビーン)が決まる。

※バッキングビーン…Javaのプログラムのこと


Play Framework
・ファイルのアップロードを簡単に行うことができる
・コンパイル不要
・Eclipseなどの既存IDEをサポート


Spring Framework
・プログラムの仕様を大幅にかえることになったとしても、
依存性のあるプログラムを外部から取り入れるようになっているので、そういった変更に強い
・結合テスト(一連の処理を終えるテスト)を行えることができる
・AOPがあるため、元のソースコードに変更を加えることなく、新しい処理を追加することができる
